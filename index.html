The "Catalytic Converter" Scam

Dealer said I needed a new cat for $1,400. Threw a P0420 code and they insisted it was failed. Smelled like rotten eggs and power was down. Almost paid it until my buddy suggested checking the downstream O2 sensor first.

The Fix: Replaced the downstream O2 sensor ($89 part, 20 minutes with a socket). Cleared the code, drove 200 miles, no return. The sensor was lazy and throwing false readings. Cat was fine all along.

Shaking Like a Washing Machine

Cylinder 2 misfire on my 5.4L Triton. Shop wanted to replace all 8 coils and plugs for $950. Truck was shaking at idle and throwing P0302 consistently. Compression test showed good numbers, so it wasn't mechanical.

The Fix: Swapped coil #2 with coil #4. Code moved to P0304. Ordered one Motorcraft coil ($65) and replaced the bad one. Also found the COP boot was oil-soaked from a leaking valve cover gasket. Fixed that too with a $12 gasket. Total: $77.

Heater Circuit Drama

Upstream O2 sensor heater circuit code. Shop quoted $450 to "diagnose and replace O2 sensor, possible wiring repair." Car ran fine but wouldn't pass inspection with the CEL on. Code kept coming back immediately after clearing.

The Fix: Checked the O2 sensor fuse first - blown 15A fuse in the engine bay box. Replaced it ($3), code cleared. Fuse blew again a week later. Traced wiring and found the O2 sensor harness was rubbing against the exhaust heat shield. Wrapped with heat tape, secured with zip ties. Fixed for good.

The Hemi Tick of Death

EGR Insufficient Flow

Failed emissions test. Shop wanted to replace the EGR valve ($320) and "clean the passages" ($280). Car was pinging under load and had a rough idle when warm. Classic EGR issues on these D-series engines.
